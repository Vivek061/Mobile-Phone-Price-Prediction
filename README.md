# 📱 Mobile Price Prediction Using Machine Learning

## 📌 Project Overview

This project focuses on predicting the price range of mobile phones using Machine Learning techniques. The model analyzes various smartphone specifications such as RAM, battery power, internal memory, camera quality, processor performance, screen dimensions, and connectivity features to classify mobile phones into different price categories.

The project demonstrates the complete Machine Learning workflow including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and comparison of multiple classification algorithms.

---

## 🎯 Objective

The main objective of this project is to build an accurate Machine Learning model that can predict the price category of a smartphone based on its technical specifications.

---

## 📂 Dataset Information

The dataset contains different mobile phone features such as:

* Battery Power
* RAM
* Internal Memory
* Mobile Weight
* Front & Primary Camera
* Screen Height & Width
* Processor Cores
* Talk Time
* 4G/3G Support
* WiFi & Bluetooth Features
* Price Range (Target Variable)
